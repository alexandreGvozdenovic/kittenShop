# Kitty Shop
E-Shop avec des chatons créé avec l'ensemble des compétences que j'ai acquis durant ma formation.
## Technologies utilisées
### Frontend
- React Native (Expo)
- React Bootstrap
- React Router DOM
- Redux
- Stripe
### Backend
- Node.js (Express framework)
- Mongo DB
